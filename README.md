<h1>TASKIVO--A Todo application</h1>

<h2>Technologies Used</h2>

Flutter<br>
Dart<br>
Provider (State Management)<br>

<h2>Key Features</h2>

✅ Add new tasks dynamically

✅ Mark tasks as completed with checkbox

✅ Delete tasks using long press gesture

✅ Real-time UI updates using Provider state management

✅ Task counter that updates automatically

✅ Clean and reusable widget structure
